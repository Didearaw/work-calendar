# 📅 Календарь графика 2/2 (работа / отдых)

Простое веб-приложение, которое показывает на год вперёд (или больше) ваши рабочие и выходные дни по циклическому графику **2 рабочих дня → 2 выходных**. Красные даты — работа, зелёные — отдых.

## ✨ Возможности

- Выбор **начальной даты** (день, с которого начинается отсчёт цикла)
- Выбор **позиции** в цикле для стартового дня (1-й рабочий, 2-й рабочий, 1-й выходной, 2-й выходной)
- Отображение **до 24 месяцев** вперёд
- Адаптивный дизайн — удобно смотреть и с компьютера, и с телефона
- Автоматический пересчёт цветов при изменении параметров

## 🚀 Быстрый старт

### 1. Локальный запуск (как статический сайт)

Скачайте файл `index.html` и откройте его в любом браузере:

```bash
git clone https://github.com/ваш_логин/work-calendar.git
cd work-calendar
# Просто откройте index.html двойным кликом
```

### 2. Запуск с помощью Flask (опционально)

Если вы хотите раздавать приложение через локальный сервер:

```bash
pip install flask
python app.py
```

Затем откройте `http://127.0.0.1:5000`

> **Примечание:** основная логика работает на чистом HTML/CSS/JS, Flask нужен только для сервера.

### 3. Бесплатный хостинг (доступ с телефона)

- **GitHub Pages** – залейте `index.html` в репозиторий, включите Pages в настройках.
- **Netlify** – перетащите папку с файлом в окно браузера.

## 🖥️ Как пользоваться

1. Установите **дату начала** (день, когда вы приступили к текущему циклу).
2. Укажите, каким по счёту в цикле был этот день:
   - 🟥 1-й рабочий
   - 🟥 2-й рабочий
   - 🟩 1-й выходной
   - 🟩 2-й выходной
3. Выберите период (по умолчанию 12 месяцев).
4. Календарь перестроится автоматически (или нажмите кнопку «Показать»).

## 🧮 Логика работы

- Цикл из **4 дней**: [работа, работа, отдых, отдых]
- От начальной даты отсчитывается разница в днях, и по модулю 4 определяется цвет каждой ячейки.
- Дни до начальной даты остаются серыми.

## 🔧 Настройка под себя

- **Изменить период по умолчанию** – в файле `index.html` найдите `<select id="monthsAheadSelect">` и поменяйте атрибут `value`.
- **Сменить цветовую схему** – в CSS измените классы `.work-day` и `.rest-day`.
